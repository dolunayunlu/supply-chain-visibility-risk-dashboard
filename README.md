# Supply Chain Visibility and Risk Dashboard

This project is a decision support dashboard for monitoring supplier performance, inventory status, shipment delays, demand trends, and operational risk in a simulated multi-product supply chain network.

## Project Scope

The dashboard includes:

- Executive supply chain overview
- Supplier performance analysis
- Inventory monitoring
- Shipment tracking
- Supply chain risk scoring
- Basic demand forecasting

## Technologies

- Python
- Streamlit
- pandas
- numpy
- Plotly
- SQLite
- scikit-learn

## Dataset

This project uses a synthetic supply chain dataset created for academic demonstration purposes. The dataset includes suppliers, products, inventory records, orders, shipments, and demand data.
